# 什么是插件

## 1. 插件定义
插件是 AIFlowy 平台的**功能扩展模块**，允许Bot通过API连接外部系统和服务，突破大语言模型的固有能力限制。

## 2. 核心价值
🔌 **无缝集成** - 快速对接企业现有系统  
⚡ **实时交互** - 毫秒级API响应能力  
🛡️ **安全可控** - 权限隔离+请求审计  
🧩 **模块化设计** - 即插即用，灵活组合
