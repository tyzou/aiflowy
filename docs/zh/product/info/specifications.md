# AIFlowy 技术规格

## 前言

AIFlowy 采用透明化的产品技术规格，以及明确的价格政策，确保你在全面了解 AIFlowy 的基础上做出决策。

## 基础信息

在 AIFlowy 之前，我们已经开发了多个 AI 底座产品，包括 Agents-Flex 、Tinyflow 和 AIEditor 等，而 AIFlowy 则是基于
Agents-Flex 、Tinyflow 和 AIEditor 等产品开发的，并针对企业场景进行了优化。

|           |                                 |   
|-----------|---------------------------------|
| **项目发布**  | 2025年4月                         | 
| **许可证类型** | 基于 Apache License 2.0 有限商业许可    |
| **后端技术**  | SpringBoot/Agents-Flex/Tinyflow |
| **前端技术**  | React/Tinyflow                  |

## 自研情况

自研情况，往往反馈里一个公司的技术能力，也突出了产品在上层应用，到底层框架的控制、优化及定制能力。

|                         |                                     |   
|-------------------------|-------------------------------------|
| **SpringBoot**          | 非自研                                 | 
| **ORM 框架 MyBatis-Flex** | 自研（ Gitee 用户增长最快的 ORM 框架，190+ 位贡献者） |
| **Agents-Flex**         | 自研（Gitee GVP 项目）                    |
| **Tinyflow**            | 自研（Gitee GVP 项目）                    |
| **文档编辑 AIEditor**       | 自研（Gitee GVP 项目）                    |
